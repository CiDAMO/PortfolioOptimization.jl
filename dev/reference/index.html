PortfolioOptimization.cvar_model_min_risk — Method
cvar_model_min_risk(df; options...)

Solve the CVaR model minimizing the risk, i.e.,

min  CVaR(x)
s.to μᵀx ≥ μmin + λ * (μmax - μmin)
     ∑ᵢ xᵢ = 1, x ≥ 0,

where λ is a keyword argument (defaults to 0.0).

source
PortfolioOptimization.cvar_model_mixed — Method
cvar_model_mixed(df; options...)

Solve the CVaR model minimizing a combination of the risk and return, i.e.,

min  λ * CVaR(x) - (1 - λ) * μᵀx
s.to ∑ᵢ xᵢ = 1, x ≥ 0,

where λ is a keyword argument (defaults to 0.5).

PortfolioOptimization.markowitz_max_returns — Method
markowitz_max_returns(μ, Σ; options...)

Solve the Markowitz mean-variance model maximizing the returns, i.e.,

max  μᵀx
s.to xᵀΣx ≤ σmin + λ * (σmax - σmin)
     ∑ᵢ xᵢ = 1, x ≥ 0,

where λ is a keyword argument (defaults to 1.0).

PortfolioOptimization.markowitz_min_risk — Method
markowitz_min_risk(μ, Σ; options...)

Solve the Markowitz mean-variance model minimizing the risk, i.e.,

min  xᵀΣx
s.to μᵀx ≥ μmin + λ * (μmax - μmin)
     ∑ᵢ xᵢ = 1, x ≥ 0,

where λ is a keyword argument (defaults to 0.0).

source
PortfolioOptimization.markowitz_mixed — Method
markowitz_mixed(μ, Σ; options...)

Solve the Markowitz mean-variance model with the a mixed objective, i.e.,

min  λ xᵀΣx / Σscale - (1 - λ) μᵀx / μscale
s.to ∑ᵢ xᵢ = 1, x ≥ 0,

where λ is a keyword argument (defaults to 0.5).

PortfolioOptimization.returns — Method
r = returns(p)

Computes the returns of the asset(s) in p, one asset per column. For each asset/column j and each instant/row i, we define

\[r_{ij} = \frac{p_{i+1,j} - p_{ij}}{p_{ij}}.\]
